# Release Notes

## Prompt Forge 3.0

Prompt Forge is a native Windows desktop app for building and iterating image-generation prompts through structured visual controls instead of slow chat-based drafting.

This release focuses on a faster desktop workflow for creators who want cleaner prompt language, stronger stylistic control, and immediate prompt feedback while shaping ideas.

### Highlights

- Live prompt updates through structured visual controls
- Guided prompt building for style, mood, composition, lighting, and color
- Dual artist influence blending
- Guided negative prompt generation
- Local presets for repeat workflows
- Native Windows desktop experience for longer prompt-building sessions

### Creative Scope

Prompt Forge includes structured controls for style, realism, texture, symbolism, camera framing, motion, atmosphere, lighting, color, output preferences, and more.

Its built-in artist influence system helps shape:

- Composition
- Palette
- Surface character
- Mood

Prompt Forge supports an enormous built-in prompt space before custom user style direction is even added, making the practical creative range effectively open-ended.
