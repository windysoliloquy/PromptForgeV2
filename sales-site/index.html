Native Windows prompt workstation

Give structure to visual intent before the model ever sees it.

Prompt Forge is a native Windows app for controlled prompt construction through deliberate visual controls. Built for people who would rather direct the image than beg for it.

It is actively developed around what the user base needs, and some image models will still rewrite parts of a copied prompt to fit their own behavior. That is expected when the app can structure an enormous prompt space (2.44 quintillion meaningful prompt variations can be added to your prompt) before you even enter a subject.

Get good at using Windy Soliloquy's Prompt Forge, and you can produce high-quality prompts up to 10 times faster than most LLMs can give you a single one.

Medium is not empty; it is restraint.

Prompt Forge uses slider states, and Medium means balanced. The goal is not to push every control to an extreme. Shape the choices that matter, leave the rest centered, and give the model a clean direction instead of a pile of competing instructions.

How it works.

Manual fulfillment note: Manual fulfillment is intentional: it activates the app cleanly, opens a direct connection with the user base, and invites serious image creators into the Prompt Forge community.

Quick clarity.

Public installer, capped demo, paid unlock, then manual delivery and import. Direct by design.

Is the installer public?

Yes. Anyone can download the signed installer. Download is public; activation is paid.

What does App Unlock do?

App Unlock removes the demo prompt-copy cap. The upgrade path is available from inside Prompt Forge.

How do I activate it?

Upgrade in the app, send your machine request code, receive the unlock file, then import it in the app.

Is there a tutorial track?

No formal tutorial track at launch. The interface is meant to be explored, and the best results come from judgment, experimentation, and taste.

Is the installer signed?

Yes. The current Windows installer is signed before release and distributed from the official GitHub release.

Where does premium fit?

Premium exists for people who want more, but it is not required to download, test, or upgrade the core app.

Why is unlock manual?

Manual fulfillment lets each paid user activate cleanly, receive the correct unlock file, and connect with the Prompt Forge community instead of being pushed through a faceless checkout.
